== Description ==

PDF Addon for [Metform](https://wordpress.org/plugins/metform/). It will automatically generate PDF files based on what users submit on the forms (metform).

Go to wp-admin > metform > PDF > install fonts.
The fonts will be downloaded automatically from the following location: [https://github.com/arshidkv12/ttfonts](https://github.com/arshidkv12/ttfonts) and placed into the 'wp-content/uploads/pdf4metform/ttfonts/' folder.
Then go to entries and select entry. Now you can see PDF file from entry details page (PDF box - bottom right)

[MetForm XLSX Exporter](https://wpdebuglog.com/downloads/metform-excel-spreadsheet/)


= PDF For Metform = 
* Automated PDF generation based on form submissions
* No need any extra configuration


== Installation ==

1. Download and extract plugin files to a wp-content/plugin directory.
2. Activate the plugin through the WordPress admin interface.
3. Go to wp-admin > metform > PDF > install fonts
4. Done !
